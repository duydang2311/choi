# choi.

a tauri music player app.

## Prerequisites

https://tauri.app/v1/guides/getting-started/prerequisites

## Developing

1. Installed npm dependencies

```bash
pnpm install
```

2. Start your first development build. Tauri would install Rust dependencies if needed.

```bash
pnpm tauri dev

# or browser only
pnpm run dev
```

## Building

```bash
pnpm tauri build
```
